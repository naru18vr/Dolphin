# ドルフィン帰宅バス

ドルフィン鍼灸院（東京都葛飾区奥戸3丁目25-14）から、周辺バス停までの徒歩ルートと、京成バス公式時刻表をスマートフォンで開くためのウェブアプリです。

## 時刻表の扱い

- このリポジトリには第三者の時刻表データを保存・再配布しません。
- 駅探の時刻表ページには、データの加工・再利用・再配布には許可が必要である旨が明記されています。そのため、駅探を取得してJSON化する処理と自動更新は停止しました。
- 行き先を選ぶと、該当するバス停の京成バス公式時刻表を開きます。公式ページで系統、行き先、のりば、当日の時刻を確認してください。
- アプリ内の曜日区分は日本時間（Asia/Tokyo）で判定し、内閣府公表の2026・2027年の国民の祝日、振替休日、国民の休日を休日として扱います。

## 開発時の確認

```sh
node --test tests/holiday.test.mjs
python3 scripts/validate_timetables.py
```

時刻をアプリ内で表示する機能を再開するには、京成バスまたは正規データ提供者から、保存・加工・再配布を許可するGTFS/APIまたはライセンスを取得する必要があります。
