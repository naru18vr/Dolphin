# ドルフィン帰宅バス

ドルフィン鍼灸院（東京都葛飾区奥戸3丁目25-14）から、周辺のバス停と各駅方面の便をスマートフォンで確認するためのウェブアプリです。

## 主な機能

- 金町・亀有・青砥・小岩・京成小岩・新小岩方面の候補を表示
- 全候補を到着の早い順に比較
- 五丁目住宅・奥戸三丁目・奥戸六丁目までの徒歩ルート地図
- Googleマップの徒歩ナビゲーションへのリンク
- 京成バス公式時刻表へのリンク

> アプリ内の便時刻は操作確認用の目安です。実際の運行時刻・遅延・のりばは乗車前に公式情報で確認してください。

## 時刻表データの更新

- 取得元は駅探の掲載路線バス時刻表です。曜日ボタンの実通信を確認し、dw=0（平日）、dw=1（土曜）、dw=2（休日）をURLに付けて3種類を個別取得します。
- 取得先ページの行き先セレクトとタイトルを照合して、d1/d2 の方向を判定します。同じ停留所・行き先・系統・方向の中で時刻を重複登録しません。
- 生成データは一時ファイルに書き、必須キー、HHMM形式、昇順、曜日別の合計便数を検証してから data/timetables.json を置き換えます。失敗時は前回ファイルを残します。
- GitHub Actionsは毎日実行し、更新前に tests/fixtures/ekitan_route.html を使った曜日・行き先・時刻抽出テストを実行します。
- アプリの「到着予定」は、出発時刻に標準所要時間を加えた概算です。道路状況・乗降時間で変動するため、乗車前は掲載元・京成バス公式ページを確認してください。
