# ドルフィン帰宅バス

ドルフィン鍼灸院（東京都葛飾区奥戸3丁目25-14）から、五丁目住宅・奥戸三丁目・奥戸六丁目の各停留所まで歩き、駅へ帰るためのスマートフォン向け案内です。

## 画面で案内する内容

- 日本時間での現在時刻と平日・土曜・休日ダイヤ
- 停留所までの徒歩時間に **2分** の安全余裕を足して、間に合う便だけを候補にする
- 発車までの残り時間、発車時刻、バス所要時間、概算到着時刻
- 同じ行き先の候補を、発車順ではなく概算到着順で比較する
- 各候補から京成バス公式時刻表とGoogleマップの徒歩案内を開く

到着時刻は固定のバス所要時間から出す**目安**です。遅延・運休・臨時変更・道路状況・のりば変更は反映しません。必ず乗車前に京成バス公式時刻表を確認してください。

## 時刻表データの状態

`data/timetables.json` は、京成バス公式時刻表を管理者が手動確認した値だけを入れるJSONです。現在は `awaiting-verification`（確認済みの便時刻は未登録）であり、アプリは時刻を推測せず公式リンクだけを表示します。

未許可の第三者サイトを自動取得して保存・再配布する処理は置きません。確認済みデータを登録する前に、対象の公式ページ・停留所・系統・方向・行き先・曜日を照合してください。

```json
{
  "mode": "manual-verified",
  "dataStatus": "verified",
  "updatedAt": "2026-07-22T12:00:00+09:00",
  "verifiedBy": "確認者名",
  "source": "京成バス公式時刻表を手動確認",
  "routes": [
    {
      "stopId": "公式の停留所ID",
      "stop": "奥戸三丁目",
      "destination": "新小岩駅",
      "destinationLabel": "公式ページ上の正式行き先",
      "line": "系統名",
      "direction": "公式の方向IDまたはのりば方面",
      "walkMinutes": 5,
      "durationMinutes": 11,
      "officialUrl": "https://公式時刻表のURL",
      "times": { "平日": ["1433"], "土曜": ["1435"], "休日": ["1435"] }
    }
  ]
}
```

時刻は `HHMM`（24〜29時も可）の4桁、各曜日で昇順・重複なしにします。同じ停留所・系統・方向・行き先は別レコードに重複させません。

## 手動更新手順

1. 公式ページで、停留所・系統・方向・行き先それぞれの平日／土曜／休日を確認します。
2. 確認済みの入力ファイルを別名で作り、次のコマンドで検証後に置き換えます。空データや推測値で既存の確認済みデータを置き換えません。

   ```sh
   node scripts/promote_timetable.mjs /path/to/verified-timetables.json
   ```
3. 次を実行してから公開します。

   ```sh
   node --test tests/*.test.mjs
   node scripts/validate_timetables.mjs
   node scripts/report_timetable_diff.mjs
   ```

4. GitHub Actions の `Validate Dolphin timetable data` が成功したことを確認します。各曜日の便数、停留所数、前回との差分がログに出ます。ある曜日が0件、必須キー欠落、不正時刻、全曜日が同じ時刻表、前回比50%以上の便数減少は失敗します。
5. 公開ページで平日・土曜・休日を切り替えて、先頭便・昼間便・最終便を公式ページと照合します。照合した停留所・行き先・曜日・公式発車時刻・アプリ表示・所要時間を変更記録に残します。

## 開発時の確認

```sh
node --test tests/*.test.mjs
node scripts/validate_timetables.mjs
```

テストは、残り時間、徒歩時間と2分余裕の境界、日付またぎ、24時台、概算到着順、最終便後、祝日、データ形式を確認します。祝日データは内閣府公表の2026・2027年分を収録し、対象年外では平日扱いにせず公式確認を促します。
